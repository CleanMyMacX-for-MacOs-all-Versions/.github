## Overview

CleanMyMac X is a comprehensive utility toolbox developed by MacPaw that helps Mac users maintain peak system performance . With over 30 built-in tools packed into an award-winning design, it addresses everything from system junk removal to malware protection . The application is **notarized by Apple** and available on the Mac App Store, guaranteeing it's free from malicious components and safe for distribution .

## Key Features

### 🧹 Smart Cleanup
- **Smart Scan** - One-click all-in-one tune-up that scans for system junk, malware, and performance issues in under 10 minutes 
- **System Junk** - Removes hidden caches, logs, and temporary files that accumulate over time 
- **Large & Old Files** - Identifies and removes forgotten files taking up valuable storage space 
- **Mail Attachments** - Cleans up duplicate and outdated email attachments 

### 🛡️ Protection & Security
- **Malware Removal** - Powered by the Moonlock Engine, MacPaw's own cybersecurity technology that detects adware, ransomware, spyware, and cryptocurrency miners 
- **Privacy Module** - Clears browsing history, chat logs, cookies, and Wi-Fi connections you no longer use 
- **Real-time Monitoring** - Keeps your system protected from emerging threats with regularly updated malware databases 

### ⚡ Performance Optimization
- **Maintenance Scripts** - Runs essential system routines including temporary file removal, log rotation, and system database rebuilding 
- **RAM Management** - Frees up memory when your system slows down 
- **Disk Repair** - Verifies and repairs disk permissions and startup disks 
- **DNS Cache Flush** - Resolves network connectivity issues 
- **Mail Acceleration** - Reindexes Mail database for faster search and better performance 

### 📱 Application Management
- **Uninstaller** - Completely removes applications and their leftover files 
- **Updater** - Keeps all your apps current with the latest versions 
- **App Reset** - Restores broken applications to their default state 

### 📊 Storage Management
- **Space Lens** - Visual map of your storage showing what's consuming disk space 
- **My Clutter** - Detects duplicates, similar photos, forgotten downloads, and large files 

### 📈 Menu App
- Real-time system monitoring from your menu bar 
- Track CPU load, memory usage, storage availability, and network speed 
- Quick access to RAM freeing, Trash emptying, and other essential tasks 

## System Requirements

- **macOS 10.13 High Sierra through macOS Tahoe 26** - Full feature support 
- **macOS 10.10-10.12** - Limited support with earlier versions (4.12.3 for Sierra, 4.10.7 for Capitan/Yosemite) 
- **macOS 10.9 and earlier** - Not supported 
- **Storage** - 210 MB free space 
- **Administrator privileges** required for full functionality

## ⚠️ Important Update

MacPaw has **ended active development of CleanMyMac X for future macOS versions** . The app remains fully functional up to **macOS Tahoe 26**, which is the last major macOS version officially supported . Existing licenses continue to work on supported systems, but the app won't receive new features or support for future macOS versions .

For users on **macOS 11 and later**, MacPaw has released a completely redesigned version simply called **CleanMyMac**, rebuilt from the ground up for modern systems and Apple Silicon .
